# 詐騙訊息預測 (Fraud Message Detection)

## 專案簡介
這個專案使用 BERT 模型來預測訊息是否為詐騙，目標是提升自動化辨識詐騙訊息的準確度，方便用於訊息過濾或警示系統。

## 功能 / 特色
- 使用 BERT 進行文字分類
- 可預測單則訊息是否為詐騙
- 利用 Hugging Face 套件快速載入模型與 tokenizer
- 結合 AI 輔助撰寫程式，加速開發流程

## 技術棧 / 工具
- 語言：Python
- 框架 / 套件：Transformers (Hugging Face), PyTorch, scikit-learn
- 其他工具：Git, GitHub, Colab
